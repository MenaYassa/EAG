# Contributing to EAG

Thank you for your interest.

## Development Philosophy

-   Architecture first
-   Small focused pull requests
-   Tests required
-   Documentation updated with every feature

## Coding Standards

-   Python 3.12+
-   Type hints
-   Ruff
-   Black
-   Pytest

## Workflow

1.  Fork
2.  Branch
3.  Implement
4.  Test
5.  Update documentation
6.  Submit PR

## Commit Style

feat: fix: docs: refactor: test: chore:

## Code Review

All changes should preserve the Constitution.
